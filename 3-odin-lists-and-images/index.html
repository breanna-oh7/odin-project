Target Attribute

  • target specifies where linkedd source will be open
  • By default, it takes the _self value, which opens the link in the current tab
  • _blank opens the link in a new tab

Rel Attribute

  • Describes relation between current page and linked doc
  • noopener value prevents opened link from gaining access to the webpage from which it was opened
  • This prevents phishing attacks
  • noreferrer value prevents opened link from knowign which weppage has the link referring to it
  • Both values are for security.
